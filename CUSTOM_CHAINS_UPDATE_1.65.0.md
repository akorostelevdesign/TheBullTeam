# Обновление 1.65.0 - Пользовательские связки дополнительных продаж

## Дата: 2026-02-14

## Что нового

### Создание пользовательских связок

Теперь вы можете создавать свои собственные связки дополнительных продаж!

**Как создать:**
1. Откройте Настройки → Дополнительные продажи
2. Нажмите кнопку "Создать связку"
3. Заполните форму:
   - **Название** - как будет называться связка (например: "Кофе с десертом")
   - **Описание** - краткое описание (необязательно)
   - **К чему предлагать?** - выберите блюдо-триггер из меню
   - **Что предлагать?** - добавьте блюда для предложения:
     - Из меню - выберите существующее блюдо
     - Свой вариант - создайте кастомное блюдо (название, описание, код R_keeper, цена)
4. Нажмите "Создать"

### Как это работает

Когда официант добавляет блюдо-триггер в заказ, автоматически появляется модальное окно с предложением добавить связанные блюда.

**Пример:**
- Триггер: "Эспрессо"
- Предложения: "Чизкейк баскский", "Медовик", "Шоколадный фондан"

### Управление связками

В разделе "Пользовательские связки" вы можете:
- ✅ Включать/выключать связки
- 🗑️ Удалять ненужные связки
- 👁️ Просматривать детали (раскрыть карточку)

### Свои варианты блюд

Если нужного блюда нет в меню, создайте "Свой вариант":
- Название (например: "Мята")
- Описание (например: "Трава для чая")
- Код R_keeper (например: "343")
- Цена (необязательно)

Это полезно для добавок, специй, или временных позиций.

## Технические детали

### Структура данных

Пользовательские связки хранятся в `dishChainSettings.custom`:

```javascript
{
  custom: {
    "кофе_с_десертом": {
      name: "Кофе с десертом",
      description: "Предложение десерта к кофе",
      triggerDish: {
        name: "Эспрессо",
        rkeeper: "123"
      },
      suggestedDishes: [
        {
          name: "Чизкейк баскский",
          rkeeper: "114",
          price: "400 рублей",
          isCustom: false
        },
        {
          name: "Мята",
          rkeeper: "343",
          price: "50 рублей",
          description: "Трава для чая",
          isCustom: true
        }
      ],
      enabled: true
    }
  }
}
```
